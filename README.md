### 🚀 Hakkımda

* 🔭 **Şu an ne üzerinde çalışıyorum:** Kendi geliştirdiğim **NovaCleanArch** framework'ü ile kurumsal projeleri standardize ediyorum.
* 🧠 **Yapay Zeka:** Geliştirme süreçlerimi `.agent`, `.grok` ve AI destekli workflow'lar ile optimize ediyorum.
* 🏗️ **Mimari Odak:** Clean Architecture, CQRS, MediatR ve Domain-Driven Design (DDD).
* ⚡ **Motto:** *"Good software is not the one that works. It's the one that keeps working under pressure."*
